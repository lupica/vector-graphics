== Description ==

Vector Graphics is a simple, lightweight plugin for adding Scalable Vector Graphics (svg) support to your Wordpress website. Allowing you to upload SVG files to your media library the same way you add any other image files. 
	
== Installation ==
	
Upload the Vector Support plugin to your site.
Activate it in your plugin directory.
That's it. Upload your vector files to your media library!

==NOTICE==

Please understand the risk of SVG/XML vulnerabilities. Always use best-practices for uploading to the web. Understanding that uploading any file to the system is a potential security risk, it is strongly recommended to only let trusted users to have upload privileges. For more information see http://security.stackexchange.com/questions/11384/exploits-or-other-security-risks-with-svg-upload
